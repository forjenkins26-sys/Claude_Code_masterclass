agent-factory-cli

Three AI agents for Playwright.

RCA · Flaky Detector · Self-Healing Locator. One swappable LLM (Groq, DeepSeek, Ollama). Orchestrated by Claude Code in production. Designed for The Testing Academy QA Masterclass.

Quickstart

From clone to first verdict

# 1. install the CLI globally
npm i -g agent-factory-cli

# 2. point it at a Playwright report
agent-factory-cli rca ./test-results/results.json

# 3. or run all three at once
agent-factory-cli triage ./test-results/results.json ./runs

Configure your LLM

# Groq (recommended free tier)
LLM_MODE=live
LLM_BASE_URL=https://api.groq.com/openai/v1
LLM_API_KEY=gsk_xxx
LLM_MODEL=llama-3.3-70b-versatile

# Or DeepSeek
# LLM_BASE_URL=https://api.deepseek.com
# LLM_MODEL=deepseek-chat

# Or Ollama (local, no key)
# LLM_BASE_URL=http://localhost:11434/v1
# LLM_MODEL=qwen2.5-coder:7b
